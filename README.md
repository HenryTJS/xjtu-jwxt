# xjtu-jwxt

针对西安交通大学教务系统的一些自动化爬取脚本。

## 脚本说明

| 脚本 | 说明 |
|------|------|
| [`all.py`](all.py) | 交互式多功能爬虫，支持 6 种查询功能 |
| [`batch_plan.py`](batch_plan.py) | 批量爬取脚本：自动遍历所有培养方案，爬取全部课程数据 |

## 功能（all.py）

| 编号 | 功能 | 说明 | 输入参数 |
|------|------|------|----------|
| 1 | **查询整体培养方案列表** | 按年级查询，自动翻页合并输出 | 年级代码（示例：2025） |
| 2 | **查询具体培养方案课程** | 需提供 PYFADM 参数 | PYFADM |
| 3 | **查询整体课表列表** | 按学期查询，自动翻页合并输出 | 学期代码（示例：2025-2026-3） |
| 4 | **查询具体教学班课表详情** | 需提供教学班 ID，学期代码从 ID 前 9 位自动提取 | 教学班ID（示例：202620271MACH40460101） |
| 5 | **查询课程列表** | 整体课程查询，自动翻页合并输出 | 无需输入 |
| 6 | **查询课程详情** | 根据课程号查询 | 课程号（示例：JAPN540912） |

> **功能 1、3、5** 支持自动翻页：每页按 999 条拉取，自动合并所有页数据输出为一个 CSV 文件。

## 批量爬取（batch_plan.py）

自动完成以下流程：

1. 输入年级代码，查询该年级**所有培养方案列表**
2. 从列表中提取每个培养方案的 `PYFADM` 参数
3. 遍历每个 `PYFADM`，爬取对应的**全部课程数据**
4. 为每条课程记录添加 `PYFAMC`（培养方案名称）列
5. 合并输出为 **`{年级代码}.csv`**（如 `2025.csv`）

```bash
python batch_plan.py
```

## 快速开始

### 1. 安装依赖

```bash
pip install requests python-dotenv
```

### 2. 配置 Cookie

1. 复制 `.env.example` 为 `.env`（**注意：`.env` 文件包含敏感信息，不会上传到 Git**）：

```bash
copy .env.example .env
```

2. 打开 `.env` 文件，将 `COOKIE_STR` 替换为你从浏览器复制的**完整有效 Cookie 字符串**。

> **如何获取 Cookie？**
> 1. 使用 Chrome/Edge 浏览器登录 [西安交通大学教务系统](https://jwxt.xjtu.edu.cn)
> 2. 按 `F12` 打开开发者工具 → 切换到 **Network（网络）** 标签
> 3. 刷新页面，任意点击一个请求
> 4. 在请求头（Request Headers）中找到 `Cookie:` 字段
> 5. 复制其完整值（**单行字符串，不要包含换行**）

### 3. 运行

```bash
python all.py
```

根据交互式菜单选择功能，输入对应参数即可。

## 输出说明

- 所有查询结果均输出为 **CSV 文件**（UTF-8 BOM 编码，可直接用 Excel 打开）
- 文件名格式：
  - `all.py`：`{功能前缀}_{标识}.csv`（如 `overall_plan_2025.csv`、`specific_schedule_{JXBID}.csv`）
  - `batch_plan.py`：`{年级代码}.csv`（如 `2025.csv`）
- 自动翻页功能会将所有页数据合并输出到**一个 CSV 文件**
- 输出文件已在 `.gitignore` 中配置，**不会上传到 Git**

## 注意事项

- Cookie 有时效性，失效后请重新从浏览器复制更新到 `.env` 文件
- Cookie 必须是一行连续的字符串，不要包含换行或多余空格
- 如果请求失败，请检查 Cookie 是否有效
- 程序会自动清理 Cookie 字符串中的空白字符和引号，兼容常见的粘贴格式
